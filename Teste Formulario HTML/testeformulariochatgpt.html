Objetivo:

Criar formulário contendo os seguintes campos:

  • Nome Completo
  • E-mail
  • Idade
  • Para qual matéria você está se candidatando para lecionar: (matemática, português, biologia, ciências, história, geografia ou artes)
  • Turnos disponíveis para lecionar
  • Nível de escolaridade: Ensino superior completo, Cursando mestrado, Mestrado completo, Cursando doutorado, Doutorado completo.
  • Descrição sobre si mesmo (máximo 200 caracteres)

Regras:

O formulário deve ter:

  • O Formulário deve ter ID e nome de "FORM"
  • Os campos devem ser, respectivamente: NOME, EMAIL, IDADE, ASSUNTO, TURNOSDISPONIVEIS, ESCOLARIDADE, DESCRICAO
  • O campo com as opções de matérias deve ter um ID também.
  • Código precisa ser semântico e com acessibilidade
